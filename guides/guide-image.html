1. JPG ‚Äî best for photos

JPG uses lossy compression but preserves photographic detail efficiently. Not ideal for text or sharp edges.

2. PNG ‚Äî best for transparency & UI graphics

PNG is lossless and supports alpha transparency. Great for logos, UI icons and illustrations.

3. WEBP ‚Äî modern balance of quality + small size

WEBP compresses much better than JPG/PNG while keeping good quality. Supported by all modern browsers.

4. AVIF ‚Äî highest quality per file size (advanced)

AVIF is extremely efficient but slower to encode. Great for photography-heavy websites.
